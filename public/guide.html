Gemini APIキーの取得方法

所要時間: 約2分
議事録AIを使うには、GoogleのAI「Gemini」のAPIキーが必要です。
Googleアカウントがあれば無料で取得できます。以下の手順に沿って進めてください。

1
Google AI Studio を開く

下のリンクをクリックして、Google AI Studioを開いてください。

🌐 https://aistudio.google.com/apikey

Googleアカウントでのログインを求められたら、ログインしてください。

2
「APIキーを作成」をクリック

ページが開いたら、右上の「APIキーを作成」をクリックします。

aistudio.google.com/apikey

API keys

↓
+ APIキーを作成

英語表示の場合は「Create API key」と書かれています

3
APIキーをコピー

キーが生成されたら、コピーボタンをクリックしてコピーします。

aistudio.google.com/apikey

あなたのAPIキー:

AIzaSyD...xxxxxxxxxxxxxxxx
↓
コピー
注意
APIキーは他人に教えないでください。あなた専用の鍵です。
4
議事録AIにキーを設定

議事録AIの画面に戻り、右上の 🔑 APIキー未設定 ボタンをクリック。

表示されたダイアログに、コピーしたAPIキーを貼り付けて「確認」ボタンを押します。

「有効なAPIキーです」と表示されれば完了です！

ポイント
APIキーは一度設定すれば、ブラウザが記憶します。次回以降は入力不要です。

設定完了！

これで議事録AIが使えます。
音声ファイルをアップロードして、議事録を自動生成しましょう。

大量に使う場合でも、従量制で非常に安価です。
